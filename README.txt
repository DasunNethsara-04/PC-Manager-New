Required Modules and Installing commands:

> Pillow
> PSutil
> wmi

Before getting started, you need to install all of these modules to run the script. To do it, open "Command Prompt" (Windows) or "Terminal" (Linux Destro.) from your operating system. Then type and run these commands

pip install pillow
pip install psutil
pip install wmi



Now its ready to run the "PC Manager v2.65" main Python script!
